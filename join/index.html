Joining the diversity book club as community!

In order to join the diversity book club as a community please get in contact with us by opening a ticket on github.

Please note the following requirements need to be met:

  • Availability of moderators (with expertise in diversity & inclusion topics)
  • Code of Conduct (with a reporting process)

If your community currently does not meet these requirements please still get in contact and we can see how we can address the issues.
